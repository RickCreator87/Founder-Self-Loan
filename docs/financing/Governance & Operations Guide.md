## Purpose
Defines how the Founder Loan is managed, documented, updated, and governed.

## Roles & Responsibilities
| Role | Responsibilities |
|------|------------------|
| Founder | Provides capital, approves forgiveness |
| LLC | Uses funds responsibly, updates ledger |
| Ledger Maintainer | Updates ledger after each event |
| Governance Reviewer | Ensures compliance & documentation integrity |

## Required Documents
- Promissory Note  
- Founder Resolution  
- Loan Ledger  
- Amendments  
- Forgiveness events  

## Operational Procedures
### Disbursements
Logged immediately with source of funds.

### Repayments
Only when profitable.

### Forgiveness
Must reference milestone or justification.

### Credit Reporting
Optional but must match ledger.
